Mentorship Opportunity: Build the Future of KubeStellar Onboarding

February 11, 2026 · One min read
KubeStellar Community
CNCF Sandbox — Multi-cluster configuration management

Opportunity Summary​

This LFX Mentorship project focuses on KubeStellar Documentation and Self-Service Enablement—creating a frictionless path for new users to adopt KubeStellar.

Timeline​

LFX Mentorship 2026 Term 1 (March–May) — 12 weeks

  • Weeks 1–2: Audit deployments and document friction points
  • Weeks 3–4: Draft Quickstart + run first user tests
  • Weeks 5–6: Build and publish interactive Killercoda tutorial
  • Weeks 7–8: Produce and publish a 5–7 module video series
  • Weeks 9–10: Create a 20-issue troubleshooting guide
  • Weeks 11–12: Deliver use-case docs + migration guides

What You’ll Build​

Deliverables include:

  • A ≤ 30-minute Quickstart
  • A hands-on interactive tutorial (Killercoda)
  • A video series (beginner → advanced)
  • A troubleshooting guide with error signatures and fixes
  • Use-case patterns and migration playbooks

Who Should Apply​

This is a strong fit if you enjoy:

  • Turning complex systems into clear learning journeys
  • Writing docs that reduce support load and improve adoption
  • Building tutorials and validation scripts
  • Working in open source communities

How to Apply​

Applications are managed through the LFX Mentorship portal:

https://mentorship.lfx.linuxfoundation.org/

Get Involved​

  • Join the KubeStellar community on CNCF Slack (#kubestellar-dev)
  • Browse issues and discussions on GitHub
  • Try the prototype site and open feedback issues
