Welcome to the KubeStellar Documentation Initiative

Welcome to the KubeStellar Documentation and Self-Service Enablement prototype demonstration site. This site showcases the vision and deliverables for the LFX Mentorship 2026 Term 1 project focused on transforming how users learn and adopt KubeStellar.

About This Project‚Äã

This initiative aims to create a frictionless onboarding experience for KubeStellar, reducing the time-to-first-success from hours to just 30 minutes. Through comprehensive documentation, interactive tutorials, video series, and migration guides, we're making multi-cluster Kubernetes management accessible to everyone.

What is KubeStellar?‚Äã

KubeStellar is a CNCF Sandbox project that simplifies multi-cluster Kubernetes management. It enables deployment and configuration of applications across multiple Kubernetes clusters with a single-cluster-like experience.

Key Benefits‚Äã

  • Single Control Plane: Manage multiple clusters from one interface
  • Native Kubernetes Objects: Work with standard Kubernetes resources without wrapping
  • Policy-Driven: Use BindingPolicies to control workload placement
  • Edge-Ready: Perfect for edge computing, multi-cloud, and hybrid deployments
